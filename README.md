# 🚨 Cyber Crime Intelligence System (India)

## Overview
This project analyzes and predicts cybercrime trends in India using Machine Learning and Data Visualization.

## Key Features
- Automated dataset download (Kaggle API)
- Data cleaning & feature engineering
- Multiple ML models (Linear Regression, Random Forest)
- Interactive dashboard (Streamlit)
- Financial loss analysis

## Insights
- Cybercrime is increasing year by year
- Financial loss is growing rapidly
- Data-driven prediction helps in cyber defense planning

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Future Scope
- Real-time API integration
- Deep Learning (LSTM)
- Crime heatmap visualization
